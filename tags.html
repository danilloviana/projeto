Descobrindo tags HTML

As tags são elementos básicos que constituem a base de todo documento HTML. Se vê esses elementos no código fonte de todas as páginas da web, logo em seguida da declaração de doctype, que está na primeira linha da página. O doctype específica qual a versão de (X)HTML que a página está usando. Os elementos que formam o conteúdo da página são postos entre a tag de abertura <html> e a tag de fechamento </html>. O elemento <html> é também conhecido como o elemento raiz.

Agora serão apresentadas algumas tags que ainda não foram vistas nas aulas:

<var></var>

Esta tag representa uma variável em uma expressão matemática ou um contexto de programação.

Exemplo de utilização: A questão matemática: <var>x</var> = <var> y </var> + 2.

<audio></audio>

Esta tag é utilizada para embutir conteúdo de som em um documento HTML ou XHTML.

Exemplo de utilização: <audio src="http://developer.mozilla.org/@api/deki/files/2926/=AudioTest_(1).ogg" autoplay>.

<wbr></wbr>

Esta tag representa uma posição no texto onde o navegador pode, opcionalmente, quebrar uma linha, embora suas regras de quebra de linha de outra forma não criar uma ruptura naquele local.

Exemplo de utilização: Para aprender AJAX, você deve estar familiarizado com o XML <wbr>Http</wbr>Request Object.

<u></u>

Esta tag define algum texto que não está articulado e tem um estilo diferente do texto normal.

Exemplo de utilização: Este é algum texto <u> mal escrito </u>.
